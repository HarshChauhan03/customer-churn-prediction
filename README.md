# 📊 Customer Churn Prediction

## 📌 Overview

This project predicts whether a customer is likely to leave a company based on customer information such as age, tenure, and account balance.

Customer churn prediction helps businesses improve customer retention strategies.

---

## 🎯 Objective

- Predict customer churn
- Learn classification algorithms
- Understand business analytics use cases

---

## 🧠 Workflow

```text
Customer Data
↓
Data Preprocessing
↓
Logistic Regression
↓
Churn Prediction
```
